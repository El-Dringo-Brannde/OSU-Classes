    Translate Grammar into Haskell Data Types

    1. Basic nonterminal -> Choose Built in type (Int, Bool)
    2. Other nonterminal -> define a data type
    3. For each production -> define Data constructor
    4. Nonterminal to constructor -> args to constructor
