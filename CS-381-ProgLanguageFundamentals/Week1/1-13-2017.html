What is a Haskell Function?

  • A black box that can repeatedly produce the same output with the same input
    • In Haskell you can only write pure functions
  • Why other languages aren't pure functions
    • Output depends on the environment
    • Could even not give an output it might just throw an exception.

    • You can't update variables in a Haskell function

    Haskell Syntax

    '_' in Haskell means match anything I don't care about the name

    '.' Composes two functions together and produces an output
